## Motivação

Escolhi o diagrama de entidade-relacionamento pois encontrei neste modelo uma forma dinâmica de demonstrar os meus conhecimentos acerca do fluxo da despesa pública, evidenciando a minha compreensão correta das dependências entre etapas através da correta definição das cardinalidades dos relacionamentos entre tabelas.

## Ferramentas/Plataformas

Como suporte ao desenvolvimento desse diagrama, foi utilizada a ERD Tool do pgAdmin 4, que realiza a engenharia reversa do schema selecionado.
A plataforma utilizada para a modelagem foi o Lucidchart[https://www.lucidchart.com/pages/pt], com o uso de um modelo específico de diagrama entidade-relacionamento.
